# Web-Based Face Recognition Attendance System
A smart attendance management system for Students, Teachers, and Admins using AI and Computer Vision.

## Key Features
* Real-time Face Scanning: Automated face detection for attendance marking.
* Role-Based Dashboards: Specific interfaces for Admins, Teachers, and Students.
* Analytics: Visual data insights for efficient classroom management.
* Cloud Database: Real-time data storage and management.

## Tech Stack
* Frontend: React.js, Tailwind CSS
* Backend: Node.js
* Database/Auth: Supabase
* AI Engine: Face-API.js
