# Face Recognition Attendance System
A smart, web-based face recognition attendance system designed for Students, Teachers, and Admins. This project automates classroom management using real-time face scanning and dashboard analytics.

# Features
Real-time Face Scanning
High-accuracy face detection and recognition using Face-API.js.
Automatic Attendance
Marks attendance instantly once the face is verified.
Role-Based Access
Separate dashboards for Admin, Teachers, and Students.
Dashboard Analytics
Visual representation of attendance trends and data.
Secure Storage
User data and logs are securely managed with Supabase.
